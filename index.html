About The Project

This project implements an AI-powered calendar application with a three-panel layout as specified. It uses React, Next.js, and TypeScript on the frontend, with a set of API routes providing backend functionality.

Core Components

  • Calendar Panel: Displays events on a monthly calendar view.
  • Chat Panel: An interactive chat interface to schedule and modify events using natural language.
  • Habit Panel: A dedicated space to track and manage personal habits.

Running the Application

  1. Ensure you have Node.js v20+ installed.
  2. Install dependencies: npm install
  3. Configure your .env file from .env.example.
  4. Run database migrations: npx prisma migrate dev
  5. Start the development server: npm run dev

Once running, the application will be available at http://localhost:3000.
